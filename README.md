# FNP-Sales-Analysis (Microsoft Excel)

# 1. Project Overview

The FNP Sales Analysis Dashboard is an interactive dashboard developed in Microsoft Excel to analyze sales performance, customer purchasing behavior, and operational efficiency. It provides a centralized view of key business metrics through dynamic visualizations and interactive filters, enabling stakeholders to monitor performance, identify trends, and make data-driven business decisions.

# 2. Purpose
The primary purpose of this dashboard is to:

-Monitor overall sales performance through key performance indicators (KPIs).
-Analyze revenue across different occasions, product categories, and time periods.
-Track monthly order trends and customer spending patterns.
-Identify top-performing products and cities.
-Evaluate average order delivery time to assess operational efficiency.
-Enable interactive analysis using filters for order date, delivery date, and occasion.

# 3.Key Features
Interactive Excel dashboard with dynamic visualizations.
KPI cards displaying Total Revenue, Total Orders, Average Customer Spend, and Average Delivery Time.
Revenue analysis by occasion, product category, and month.
Monthly order quantity trend analysis.
Top 10 cities by order volume.
Top 5 products by revenue.
Customer gender distribution analysis.
Interactive slicers for filtering data by Order Date, Delivery Date, and Occasion.
Clean and user-friendly dashboard layout for quick business insights.
